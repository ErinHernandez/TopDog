# Slow Draft Sandbox

A self-contained sandbox environment for developing and testing SlowDraft components.

## Structure

```
sandbox/slowdraft/
├── components/          # SlowDraft component files
│   ├── SlowDraftCard.tsx
│   ├── FilterSortBar.tsx
│   ├── MyRosterStrip.tsx
│   ├── NotablePicks.tsx
│   ├── PositionNeedsIndicator.tsx
│   └── index.ts
├── hooks/               # Custom hooks
│   ├── useSlowDrafts.ts
│   └── index.ts
├── deps/                # Dependency stubs
│   ├── core/
│   │   └── constants/
│   │       ├── colors.ts
│   │       └── sizes.ts
│   ├── draft-room/
│   │   └── constants.ts
│   └── components/
│       └── shared/
│           └── index.tsx
├── SlowDraftsTabVX2.tsx # Main component
├── types.ts             # Type definitions
├── constants.ts         # SlowDraft constants
├── index.ts             # Module exports
├── Sandbox.tsx          # Test page
└── README.md            # This file
```

## Components

### SlowDraftsTabVX2
Main component that displays a list of slow drafts with filtering, sorting, and card-based UI.

### SlowDraftCard
Individual draft card component with collapsed/expanded states showing:
- Tournament name and pick info
- Roster visualization
- Position needs
- Notable events
- Quick actions

### FilterSortBar
Filter and sort controls for managing multiple drafts.

### MyRosterStrip
Visual representation of drafted players as position-colored squares or cards.

### PositionNeedsIndicator
Shows position requirements and roster construction needs.

### NotablePicks
Activity feed showing reaches, steals, and other notable draft events.

## Usage

Import and use the main component:

```tsx
import SlowDraftsTabVX2 from './SlowDraftsTabVX2';

<SlowDraftsTabVX2
  onEnterDraft={(draft) => console.log('Enter draft', draft)}
  onJoinDraft={() => console.log('Join draft')}
  onQuickPick={async (draftId, playerId) => {
    console.log('Quick pick', { draftId, playerId });
  }}
/>
```

Or use the sandbox page:

```tsx
import Sandbox from './Sandbox';

<Sandbox />
```

## Dependencies

The sandbox includes minimal dependency stubs in the `deps/` directory:
- Color constants (BG_COLORS, TEXT_COLORS, STATE_COLORS, POSITION_COLORS)
- Size constants (SPACING, RADIUS, TYPOGRAPHY)
- Draft room constants (TILED_BG_STYLE)
- Shared components (Skeleton, EmptyState, ErrorState)

All imports have been updated to use relative paths from the sandbox root.

## Features

- ✅ Self-contained (no external dependencies beyond React)
- ✅ Mock data generation for testing
- ✅ Filtering and sorting
- ✅ Collapsible draft cards
- ✅ Position needs tracking
- ✅ Notable events feed
- ✅ Quick pick functionality

## Development

The sandbox uses mock data generated by the `useSlowDrafts` hook. To connect to real data, replace the mock data generation in `hooks/useSlowDrafts.ts` with actual API calls.
